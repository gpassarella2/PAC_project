Di seguito sono riportati i casi d’uso relativi al sistema di gestione percorsi e viaggi.
\begin{itemize}
    \item \textbf{UC1 - Registrazione:} Come nuovo utente, voglio potermi registrare al sistema per creare un account personale.

    \item \textbf{UC2 - Login:} Come utente registrato, voglio poter accedere al mio account per utilizzare le funzionalità del sistema.

    \item \textbf{UC3 - Logout:} Voglio poter terminare la mia sessione in modo sicuro.

    \item \textbf{UC4 - Autenticazione:} Voglio che il sistema verifichi le mie credenziali per consentire l’accesso sicuro.

    \item \textbf{UC5 - Gestione profilo:} Voglio poter gestire le informazioni del mio profilo personale.

    \item \textbf{UC6 - Modifica dati:} Voglio poter modificare i miei dati personali.

    \item \textbf{UC7 - Cambio password:} Voglio poter aggiornare la mia password per motivi di sicurezza.

    \item \textbf{UC8 - Elimina account:} Voglio poter eliminare definitivamente il mio account.

    \item \textbf{UC9 - Visualizza percorso:} Voglio poter visualizzare i dettagli di un percorso.

    \item \textbf{UC10 - Catalogo percorsi:} Voglio poter consultare un catalogo di percorsi disponibili.

    \item \textbf{UC11 - Filtri ricerca:} Voglio poter filtrare i percorsi secondo determinati criteri.

    \item \textbf{UC12 - Salva percorso:} Voglio poter salvare un percorso tra i miei preferiti.

    \item \textbf{UC13 - Visualizza percorsi salvati:} Voglio poter visualizzare l’elenco dei percorsi salvati.

    \item \textbf{UC14 - Visualizza storico:} Voglio poter visualizzare lo storico dei percorsi effettuati.

    \item \textbf{UC15 - Esporta percorso:} Voglio poter esportare un percorso in un formato condivisibile.

    \item \textbf{UC16 - Rimuovi percorso salvato:} Voglio poter rimuovere un percorso dai miei salvati.

    \item \textbf{UC17 - Condividi percorso:} Voglio poter condividere un percorso con altri utenti.

    \item \textbf{UC18 - Aggiungi recensione:} Voglio poter aggiungere una recensione a un percorso.

    \item \textbf{UC19 - Imposta percorso completato:} Voglio poter segnare un percorso come completato.

    \item \textbf{UC20 - Crea nuovo viaggio:} Voglio poter creare un nuovo viaggio personalizzato.

    \item \textbf{UC21 - Scegli percorso esistente:} Voglio poter creare un viaggio partendo da un percorso già esistente.

    \item \textbf{UC22 - Crea percorso nuovo:} Voglio poter creare un percorso completamente nuovo.

    \item \textbf{UC23 - Genera percorso casuale:} Voglio poter generare automaticamente un percorso casuale.

    \item \textbf{UC24 - Imposta punto di partenza:} Voglio poter scegliere il punto di partenza del viaggio.

    \item \textbf{UC25 - Selezione monumenti:} Voglio poter selezionare i monumenti o punti di interesse da includere nel percorso.

    \item \textbf{UC26 - Inserimento manuale tappe:} Voglio poter inserire manualmente le tappe del percorso.

    \item \textbf{UC27 - Scelta tempistiche:} Voglio poter definire le tempistiche del viaggio.

    \item \textbf{UC28 - Ottimizzazione percorso:} Voglio che il sistema ottimizzi automaticamente l’ordine delle tappe.

    \item \textbf{UC29 - Modifica tappe:} Voglio poter modificare le tappe di un percorso creato.

    \item \textbf{UC30 - Elimina tappa:} Voglio poter eliminare una tappa dal percorso.
\end{itemize}
